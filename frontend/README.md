# GLinks CR Frontend


## Instalación

### 1. Acceder a la carpeta

```bash
cd glinks-cr
```

### 2. Instalar dependencias

Con npm:

```bash
npm install
```

### 3. Instalar tailwind

```bash
npm install tailwindcss @tailwindcss/vite
```

### 4. Levantar el servidor de desarrollo

```bash
npm run dev
```
